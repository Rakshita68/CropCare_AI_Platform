# Contributing to Global Agriculture AI Platform

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/yourusername/agriculture-ai-platform.git`
3. Install dependencies: `pip install -r requirements.txt`
4. Copy `.env.example` to `.env` and add your API keys
5. Run the system: `python start_server.py`

## Development Setup

### Backend
```bash
cd backend
pip install -r requirements/requirements.txt
python -m app.main
```

### Frontend
Open `frontend/index.html` in your browser or run the complete system.

## Pull Request Process

1. Create a feature branch
2. Make your changes
3. Test thoroughly
4. Submit a pull request with clear description

## Code Style

- Follow PEP 8 for Python code
- Use meaningful variable names
- Add comments for complex logic
- Ensure all tests pass
